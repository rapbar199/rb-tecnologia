Perguntas frequentes

O app precisa de internet?

Não. O Cauro funciona totalmente offline. Tudo fica no seu aparelho.

Meus dados vão para algum servidor ou para a RB Tecnologia?

Não. Não temos servidores e não coletamos nada. Seus lançamentos ficam só no seu aparelho. Veja a Política de Privacidade.

Como faço backup dos meus dados?

Em Mais → Backup → Fazer backup. O app gera um arquivo que você salva ou compartilha onde quiser. Faça isso de vez em quando — como os dados ficam só no aparelho, o backup é a forma de não perdê-los.

Como levo meus dados para outro aparelho?

Faça o backup no aparelho antigo e use Mais → Backup → Restaurar backup no aparelho novo.

Vim do Mobills ou do Organizze. Dá para trazer meu histórico?

Sim. Exporte um CSV no outro app e importe em Mais → Importar CSV.

O Cauro é grátis mesmo?

Sim. Todo o controle de gasto é grátis e completo, sem anúncio e sem limite artificial.

Como apago tudo?

Em Mais → Seus dados → Apagar tudo, ou desinstalando o app.
